# Shaders Playground

A Next.js app for experimenting with shaders and visual effects.

## Setup

```bash
npm install
```

## Running

```bash
npm run dev
```

Open [http://localhost:3000/editor](http://localhost:3000/editor) in your browser.

## Other commands

```bash
npm run build   # production build
npm run start   # serve production build
npm run lint    # lint
```

## Stack

- [Next.js](https://nextjs.org) 16
- React 19
- Tailwind CSS 4
- [`shaders`](https://www.npmjs.com/package/shaders) — shader utilities
- [`motion`](https://motion.dev) — animations
